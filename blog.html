A. What is the Document Object Model (DOM)?

DOM (Document Object Model) is a programming interface provided by web browsers for web documents like HTML. It represents the structure of the document as a tree of objects where each node corresponds to an element, text, or other content in the document

B. How do you select an element from the DOM?

There are several methods for select an element from the dom, as : getElementById(),getElementByClassName,getElementByTagName etc.

C. What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript where we attach a single event listener to a parent element that listens for events triggered by its child elements. Instead of attaching individual event listeners to multiple child elements, we handle events centrally at the parent level by leveraging event bubbling.
